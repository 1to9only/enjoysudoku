by tarek » Sat Dec 11, 2010 10:46 pm

IMO, the non minimal version of a puzzle APPEARS more difficult than the minimal version due to the set of techniques we are using to solve puzzles.

The question will always be: How could the non minimal version be more difficult to solve when it is closer to the solution?

The "easier to solve" minimal version is contained within the "Harder to solve" non minimal puzzle! This highlights our lack of resources to find that easier path. We know that it should be easier but we can't prove it YET.

This is similar in a strange way to UR destroying puzzle information (which RW disproved) ... My feeling is that this situation is the same. I would stick with posting minimal puzzles as the most difficult version.
