Let me answer in my own way.

1) Sudoku Explainer is the referee for the pattern game.
Al players must use the same version as gremlin in that game.

2) Out of the pattern game, SE rating is not a key issue.
As SE gives a "well known" ??? rating, it's good to have it for "hardest puzzles", although this is the weakest part of the program.
For such ratings, it's better to use a version "as clean as possible", but the rating problems encountered in SE are not so common.

3) An attempt is made to produce a C++ version of SE much faster. Working on that issue, plasson and I have found other bugs in the rating that shoul be fixed as well.

I use already the FIXED3 version of SE in my tests to produce the C++ version of SE, but I'll stick for sure to the common one for the game.
