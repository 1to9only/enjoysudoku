I did it! Just now. I solved the "Golden Nugget" Sudoku by hand. I didn't find it difficult, started it yesterday afternoon and had it solved in only about three hours, I've taken a lot longer than that on a SE 9.3 puzzle. But it wasn't difficult only because I happened to be very lucky, I was just settling down to try in turn an estimated 350 trials, and hit the answer on trial 37, so could have spent ten times as long solving it.

The solution method was as follows. I started by looking for two lines that have the smallest number of alternative possibilities for the unknowns, these had to be either Row 4, Column 3 or Column 5. I've found in the past that parallel lines (both rows or both columns) work best so settled on Column 3 for the main trial with Column 5 as backup.

There are 34 possible ways to fill in Column 3, so I tried in turn all 34. This quickly eliminated 10 of them. Then, starting with the numerically smallest remaining possibility for Column 3, with 124576 reading from top to bottom, I listed all 13 possibilities for Column 5. The third of these that I happened to try, with 476312 reading from top to bottom, rapidly gave the correct answer using only elementary techniques.
