Hi champagne,

Every 10+ puzzle that can be morphed to fit in a given pattern is assigned to this pattern and is counted once. Any of the patterns is counted once.
If you sum vertically the first column, you find the total number of the patterns.
If you multiply the two columns in a row and then sum, you find the total number of puzzles.

Why you think the counts are unrealistic?
