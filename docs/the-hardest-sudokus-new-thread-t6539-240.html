Below are some statistics, once for the same 2 puzzles, and once for all puzzles from patterns game 141.
Sorry for the ugly format.
The procedure removes each given, then collects UA, and determines the cells solved w/o the help of the removed clue.
Columns are:
first row
- the source puzzle with comments
second row
- pussle
- nGivens
- sum(count(UA)) = count of UA hit by a single clue
- max(UA size)
- sum(UA size). This divided by sum(count(UA)) gives the average UA size.
- sum(solved cells)
third row
- list of max(UA size) for each clue
- sum(max(UA size)). This divided by the number of clues gives the average maximal UA size per clue.
